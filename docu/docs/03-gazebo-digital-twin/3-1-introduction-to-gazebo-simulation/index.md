### Learning Objectives

Upon completion of this chapter, students will be able to:
- Understand the fundamental role of Gazebo in robotics simulation and development.
- Install and configure Gazebo on a Linux system, particularly with ROS 2.
- Navigate the Gazebo graphical user interface and interact with basic simulation controls.
- Explain core simulation concepts such as physics engines, simulation time, and entities.
- Recognize the basic structure and purpose of the Simulation Description Format (SDF).
